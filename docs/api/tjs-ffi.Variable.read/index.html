Variable: read

const read: object

Direct memory reads from a pointer at a given byte offset. Faster than creating an intermediate buffer for one-off reads.

Note: u64/i64 return a JavaScript number, which cannot represent every 64-bit value. Magnitudes above Number.MAX_SAFE_INTEGER (2**53 - 1) lose precision, and u64 values with the high bit set read back as a negative number (the bytes are interpreted as a signed int64). Use the low 53 bits only, or read the raw bytes via toUint8Array if you need the exact value.
